# Politique de sécurité

## Versions supportées

Seule la dernière version publiée de Nebula Launcher reçoit des correctifs de sécurité.

## Signaler une vulnérabilité

Si tu découvres une faille de sécurité dans Nebula Launcher (le launcher ou le bot Discord de patch notes) :

- Utilise de préférence le [signalement privé de vulnérabilité](https://github.com/ShadowLiveDiscord/CrazyTorgnole/security/advisories/new) sur GitHub (onglet Security > Advisories > Report a vulnerability). Cela permet de discuter et corriger le problème avant qu'il ne soit rendu public.
- Merci de ne pas ouvrir d'issue publique pour une faille de sécurité non corrigée.

Donne autant de détails que possible : étapes de reproduction, version concernée, impact potentiel.

## Délai de réponse

Ce projet est maintenu sur le temps libre d'une seule personne — merci de ta patience, les retours se font au mieux selon la disponibilité.
